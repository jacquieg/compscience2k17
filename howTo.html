When adding a navbar to your webpage you have to put it inside the body tags.
A div with the id of "top of page" is added to place the navbar at the top of
your page. We also add another id of "navbar" which is used when styling and
under CSS. We use list tags to list every page that is to be linked to the webpage
you are on. 

                    
#topOfPage{
    text-align:center;
}
#navbar a{
    text-decoration:none;
    color:white;
}
#navbar li{
    display:inline;
    text-decoration:none;
    padding:20px;
    border-style:groove;

}

Under the CSS tags I added code to style my navbar. For example, I used the Id of
of "top of page" to place my navbar at the top of the my webpage, but also to
center it. I used a "navbar a" id to style the writing inside the boxes of the
navbar. By writing I mean the titles of the page like for example, "About me".
I used another id of "navbar li" to style around the titles of the pages. I
added an image behind the writing, a border around it, and just a bit of padding
to change the size of the box.

How to code a calculator

The answer is: 

                    
     <input type="text" name="input1" style="width:50px;" id="input1"/>
     <select id="dropdown">
            <option value="+">+</option>
            <option value="-">-</option>
            <option value="*">x</option>
            <option value="/">/</option>
    
     </select>
     <input type="text" name="input2" style="width:50px;" id="input2"/>
     <input type="button" value="calculate" id="submitbutton"/>
    <h2 style="color:white;">The answer is: <br> <span id="display"></span></h2>

This part of the code is put into HTMl. What it does is set our calculator up as you
see in the box above. The HTMl only creates the look or design of it but doesn't
make it function completely. As you can clearly see it gives us more than one
option for operations we can use such as division, addition, or multiplication.
We also have a "calculate" button added so we can get our final answer. Right
below it we have a space provided for our given answer. 

                    
var button = document.getElementById("submitbutton");
var input1 = document.getElementById("input1");
var input2 = document.getElementById("input2");
var display = document.getElementById("display");
var dropdown = document.getElementById("dropdown");
var answer;


button.addEventListener("click", doMath);

function doMath(){
    if (dropdown.value == "+"){
        answer = parseInt(input1.value) + parseInt(input2.value);
        display.innerHTML = answer;
    }else if (dropdown.value =="-"){
        answer = parseInt(input1.value) - parseInt(input2.value);
        display.innerHTML = answer;
    }else if (dropdown.value =="*"){
        answer = parseInt(input1.value) * parseInt(input2.value);
        display.innerHTML = answer;
    }else if (dropdown.value =="/"){
        answer = parseInt(input1.value) / parseInt(input2.value);
        display.innerHTML = answer;
    }
}

This part of our code is now added to javascript. What we do here now is what
is what makes our calculator actually function. For example, under do math
we see if and else if statements. These are what connect our inputs, or in
other words the numbers that we are trying to calculate. Like if we want the
numbers to be added together, then inside the else if statement we tell the code
to add "input1" and "input2" together.
