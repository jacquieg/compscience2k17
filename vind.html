Vindinium

What is Vindinium?
Vindinium is a video game that programmed using AI. Users must program their bots to do 3 things which are:
    .get mines
    .go to tavers
    .attack enemies

How I Coded My Bot
How My Bot Goes To Taverns At 50hp:

When it comes specifically to this, I programmed
my bot to go to the closest tavern for health
when it is under 50hp. This is so that in
situations like when it's in danger, it can stay
alive. What the code does here is it looks at the positions
of all the taverns. It the determines which is the
nearest to my Bot wherever, depending on where it is on the map
It then console.logs that it is heading towards the closest Tavern.
The matter of when it takes this action is in the if else statement
 
if(task === "tavern") {
    var closestTavern = bot.taverns[0];
        for( i = 0; i < bot.taverns.length; i++) {
            if(bot.findDistance(myPos, closestTavern) > bot.findDistance(myPos, bot.taverns[i])) {
                closestTavern = bot.taverns[i];
              }
        }
        console.log("going to taverns");
        myDir = bot.findDistance(myPos, closestTavern);
        }
How It Claims A Free Mine:
My Bot will now look at all the free mines in position.
It determines which is closer, then determines the action of going to
that mine to make it their own. My reasoning behind setting up the code this way
was so that when the game begins, I will start getting gold very instantly and rapidly
which helps me, the player, have better chances of winning the game.
 
            
if(task === "freeMines") {
    var closestMine = bot.freeMines[0];
        for(i = 0; i < bot.freeMines.length; i++) {
            if(bot.findDistance(myPos, closestMine) > bot.findDistance(myPos, bot.freeMines[i])) {
                closestMine = bot.freeMines[i];
            }
        }
        console.log("claiming free Mines");
        myDir = bot.findPath(myPos, closestMine);
    }
How To Attack Enemies
When there are no more free mines in sight and the bots health is higher than 50hp
it will start looking to attack enemy bots. This doesn't however mean that it looks to
attack just any bot. It looks at each bots array, then determines which is closest.
It then obviously goes towards them to eliminate them.
 
if(task === "closestEnemy") {
    var closestEnemy = bot.enemy[0];
        for( i = 0; i < bot.enemy.length; i++) {
            if(bot.findDistance(myPos, closestEnemy.posArray) > bot.findDistance(myPos, bot.enemy[i].posArray)) {
               closestEnemy = bot.enemy[i];
            }
        }
        console.log("attacking closest enemy");
        myDir = bot.findDistance(myPos,closestEnemy.posArray);
    }
What I Learned About AI:
This was not my first year having an experience working with AI or vindinium, which ever you want to
concider it, but last year I didn't get much of a feel or understanding behind the purposes or what
this was. My very first time hearing about it, it truthfully didn't interest me as much as it
should have, but this year I really tried to understand it and have as much of a good time coding
as I was playing it. Thanks to the help of a peer, Understandind the code helped me much more.
What I learned by playing and coding this was that to be a successful player you have to know
tacticts and solutions to every aspect, such as: claiming freemines, attacking enemy bots, or simply
just looking for taverns. Of course playing the game, there is always a simple way of getting it done,
but the best part of the game is being able to challenge yourself to reach something higher than what is
expected. Other players are in the same position as you are, which only makes the game much more intense.
Using these skills in real life can help in many ways as well. Personally I believe so, but we are all
different as individuals. Back to the subject, Ai and computer science are full og knowledge and getting
an experience with these has helped me grow as a student.
